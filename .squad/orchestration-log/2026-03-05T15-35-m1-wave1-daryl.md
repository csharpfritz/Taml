## Key Design Decisions

1. **Frozensets for Boolean Detection:** O(1) lookup with immutable storage. Case-insensitive matching via `.lower()`.
2. **Raw Text Index Tracking:** Uses `skip_to` index to avoid re-processing lines. Clean integration with existing stack-based parser.
3. **Date Detection with datetime Module:** Uses Python's `datetime.fromisoformat()` with Z suffix preprocessing for maximum stdlib compatibility.
4. **isinstance() for Type Detection:** No need for look-ahead; runtime `isinstance(parent[key], dict)` cleanly detects duplicate key patterns.
5. **PEP 8 Compliance:** All code maintains existing style, type hints, and docstring conventions.
